# Yard::BitStruct

YARD extension for bit-struct's DSL

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'yard-bit-struct'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install yard-bit-struct
